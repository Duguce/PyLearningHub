    🔥 项目结构

| 序号 |                       项目名称                       |                            数据集                            |                          数据集概述                          |
| :--: | :--------------------------------------------------: | :----------------------------------------------------------: | :----------------------------------------------------------: |
|  1   |     [电商零售数据分析](./电商零售数据分析.ipynb)     | [E-Commerce Data（kaggle）](https://www.kaggle.com/datasets/carrie1/ecommerce-data) | 本项目使用的数据集来自于某在线零售业务的交易数据，该跨国公司主要销售礼品，大部分销售对象是批发商。 |
|  2   | [淘宝用户行为数据分析](./淘宝用户行为数据分析.ipynb) | [UserBehavior（TIANCHI）](https://tianchi.aliyun.com/dataset/649?spm=5176.12281976.J_3941670930.10.278012c4DsfWE6) | 本项目使用的数据集是UserBehavior，它是阿里巴巴提供的一个淘宝用户行为数据集，内容主要包含了2017年11月25日至2017年12月3日之间，有行为的约一百万随机用户的所有行为。 |
|  3   | [京东订单销售数据分析](./京东订单销售数据分析.ipynb) | [JD_Order_Dataset（提取码：hhfi）](https://pan.baidu.com/s/117L4bCPIxQyWJ8qKyVnfYw) | 本项目使用的数据集是2020年5月25号京东大家电销售-冰箱订单数据，按照10%随机抽取的比例抽取，包含约70万条相关数据。 |
